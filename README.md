## Features

-   Authentication with JWT

## Installation

-   `git clone https://github.com/fajargema/book-api.git`
-   Copy .env from .env.example `cp .env.example .env`
-   Run `php artisan key:generate` and `php artisan jwt:secret`
-   `php artisan migrate`
-   `php artisan serve`
